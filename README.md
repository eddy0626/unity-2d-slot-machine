# 🎰 Unity 2D Slot Machine Game

Unity로 제작한 2D 슬롯머신 게임입니다.

## 📁 프로젝트 구조

- **01.Scenes** - 씬(Scene) 파일
- **02.Scripts** - C# 스크립트
- **03.Prefabs** - 프리팹(Prefab)
- **04.Images** - 스프라이트 이미지 및 머터리얼
- **05.Models** - 모델 파일
- **06.Sounds** - 오디오 파일
- **07.Animations** - 애니메이터 컨트롤러 및 애니메이션 클립
- **08.Fonts** - 폰트 파일

## 🛠 개발 환경

- Unity 2D
- C#
